# 📚 图书管理系统（Library Management System）

一个基于 Web 的简单图书管理系统，用于跟踪书籍、作者和阅读状态。

## ✨ 功能特色

- 添加新书籍，包括标题、作者、页数和阅读状态  
- 切换书籍的阅读状态  
- 响应式设计，提供简洁的用户体验

## 🛠️ 使用方法

1. 在浏览器中打开 `index.html` 文件  
2. 点击 “New Book +” 按钮添加新书籍  
3. 填写书籍信息并提交表单  
4. 通过点击复选框切换阅读状态

## 📁 项目结构

```

├── index.html      # 主界面 HTML 文件
├── style.css       # 页面样式文件
└── script.js       # JavaScript 逻辑

```
